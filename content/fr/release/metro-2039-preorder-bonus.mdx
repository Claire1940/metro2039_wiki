## Comment utiliser votre bonus de précommande Metro 2039

L'utilisation de vos objets est un processus simple, mais il varie légèrement selon la plateforme. La plupart des boutiques numériques regrouperont automatiquement le **bonus de précommande de metro 2039** avec le téléchargement de votre jeu. Cependant, si vous avez acheté une version physique, suivez ces étapes :

1.  Localisez le code promotionnel à l'intérieur du boîtier du jeu ou sur votre reçu.

En jeu, ces objets sont généralement accessibles au premier « Hub », comme une station sécurisée ou une base mobile. Cherchez une caisse d'équipement verte marquée de l'insigne spartiate pour réclamer votre équipement.
